# Directory Layout

## Key Directories

| Path | Contents |
|------|----------|
| `firmware/stock/` | Original stock firmware images (renamed from `_a` suffix) |
| `firmware/custom/` | Working images - input for AVB resign scripts |
| `output/` | Final AVB-resigned images ready to flash |
| `keys/` | AVB signing keys (vbmeta RSA4096, system AOSP testkey RSA2048) |
| `app_keystore/` | APK signing keystores (JKS format) |
| `certs/` | Exported certificates, DER hex, AOSP-format keys |
| `certs/aosp/` | AOSP-format signing keys (.x509.pem + .pk8) per seinfo type |
| `ROM_resigner/` | APK re-signing tool (ybtag fork, modified) |
| `ROM_resigner/AOSP_security/` | Default signing keys (copy of certs/aosp/) |
| `scripts/` | AVB resign scripts |
| `tools/` | avb-tools, android-bins, fec, ext4_utils, lpunpack_and_lpmake |
| `docs/` | Investigation findings and documentation |

## Signing Keys

### AVB Keys (in `keys/`)
| File | Purpose |
|------|---------|
| `vbmeta.pem` | RSA4096 - signs vbmeta |
| `vbmeta_system.pem` | AOSP testkey RSA2048 - signs system chain |
| `system.avbpubkey` | Public key for system chain partition (generated by generate_keys.sh) |

### APK Keys (in `certs/aosp/` and `ROM_resigner/AOSP_security/`)
| File | seinfo | CN |
|------|--------|-----|
| `platform.{x509.pem,pk8}` | platform | example-platform |
| `media.{x509.pem,pk8}` | media | example-media |
| `verizon.{x509.pem,pk8}` | verizon | example-verizon |
| `ssgapp.{x509.pem,pk8}` | ssgapp | example-ssgapp |
| `sysmonapp.{x509.pem,pk8}` | sysmonapp | example-sysmonapp |
| `shared.{x509.pem,pk8}` | shared | example-shared |
| `releasekey.{x509.pem,pk8}` | releasekey | example-releasekey |
| `networkstack.{x509.pem,pk8}` | networkstack | example-networkstack |

All keys are EC P-256 (secp256r1). Each seinfo type has a UNIQUE key to avoid PolicyComparator duplicate detection.
